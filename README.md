# Smart Waste Segregation AI

An AI-based waste classification system that detects waste types
such as plastic, metal, organic, and recyclable.

## Features

- AI waste detection
- Image upload
- Gamification points system
- Leaderboard
- User feedback
- Dark blue professional UI

Frontend:
HTML
CSS
JavaScript

Deployment:
GitHub Pages

## Purpose

Encouraging proper waste segregation and recycling using technology.
